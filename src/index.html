1
Channel Assignment - Assigns this parameter to an animation channel (0 = not animated, 1-5 = channel number). When assigned, the parameter will animate between start and end values.
2
Start Value Button - Click to set the current value as the animation start point. The parameter will begin at this value when the animation plays.
3
Value Slider - Drag to adjust the parameter value in real-time.
4
End Value Button - Click to set the current value as the animation end point. The parameter will reach this value when the animation completes.
5
Numeric Input - Type exact values for precise control. Synced with the slider for real-time visual feedback.

1
Timeline Slider - Shows the current playback position. Drag to seek through the animation. When the channel is playing, this slider animates automatically.
2
Channel Number - Identifies which channel this is (1-5). The color matches the color used in parameter inputs when assigned to this channel.
3
Arm Checkbox - Arms this channel for animation. When armed and you press play, parameters assigned to this channel will animate.
4
Duration - How long (in seconds) it takes to animate from start to end. The animation will yoyo back and forth continuously.
5
Easing Function - Controls the animation curve. Sin creates smooth waves, Linear is constant speed, Power/Circ create acceleration effects, Custom allows drawing your own curve.
6
Offset - Phase shift in seconds. Allows staggering multiple channels so they don't all animate in sync.
7
Clear Button - Removes all parameter assignments for this channel and resets it to default state.
8
Seek to Start [,] - Jumps all armed channels to their starting position. Keyboard shortcut: comma key.
9
Play [Space] - Starts animation playback for all armed channels. Keyboard shortcut: spacebar.
10
Pause [Space] - Pauses animation playback. Keyboard shortcut: spacebar.
11
Seek to End [.] - Jumps all armed channels to their ending position. Keyboard shortcut: period key.
12
Screenshot - Captures the current frame as a PNG image and downloads it to your computer.
13
Arm Video Export - Arms video recording. When armed, press Play to start recording an MP4 video of your animation.
